####문자열(String)
##### isnumeric()숫자인가?
```python
'1'.isnumeric()
True
'1a'.isnumeric()
False
```
##### title()첫 글자를 대문자로 바꾼다
```python
'abc'.title()
Abc
```
##### upper()대문자로 바꾼다
```python
'abc'.upper()
ABC
```
##### lower()소문자로 바꾼다
```python
'ABC'.lower()
abc
```
