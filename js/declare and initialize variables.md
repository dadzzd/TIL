변수 선언과 초기화
==================
## 변수
**프로그램 실행 도중 임의 값을 저장해 두고 읽을 수 있는 공간**
#### 변수의 선언과 초기화
+ 선언    
컴퓨터에게 변수를 사용할 것이라고 선언 역할
```js
var variable_name;
variable_name = value;
```
+ 초기화   
선언한 변수에 처음으로 값을 저장하는 과정 
+ 변수에 값을 저장하는 법 
    - 대입연산자(=)를 활용
    - 콤마(,)를 활용해 여러 변수를 동시에 선언, 초기화 가능
   
#### 변수의 활용
변수의 이름을 사용하면 변수의 값을 읽어서 사용 가능
```js
var var1 = "1", var2 = "2", var3 = "3";

console.log(var1);
alert(var2);
prompt(var3);
```
